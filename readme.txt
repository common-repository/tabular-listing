== Description ==

A plugin that help you with listing your posts in a tabular fashion as well as providing the criteria for filtering data.
Provide options to format output of post data and filter them according to selection.

Support is provided at [my blog](http://blog.centilin.com/). You can also view plugin info and download it at the [plugin page](http://blog.centilin.com/everything_it/web-design/wordpress/tabular-listing/ "Wordpress plugin: Tabular Listing")

== Installation ==

1. Upload `tabular-listing.zip` or download plugin from repository to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You are good to go
